**AI Chatbot with Gemini API**:

This is a simple AI-powered chatbot built using HTML, CSS, and JavaScript, integrated with Google's Gemini API for generating intelligent responses. The chatbot provides a user-friendly interface, allowing users to interact in real time by sending messages via a text input or a send button.

Features
✅ Interactive chatbot UI
✅ Real-time AI responses using Gemini API
✅ Click or "Enter" key to send messages
✅ Toggle chatbot visibility with an open/close button
✅ Automatic scrolling to keep the latest message at the top

Technologies Used
Frontend: HTML, CSS, JavaScript
API: Google Gemini API
